# Vite WordPress

WordPressテーマ開発用環境

## 要件

- Docker
- Node.js >=20.11.0

## 機能

- Adminer
- Mailpit
- TypeScript
- SCSS
- 画像最適化
  - AVIF / WebP 自動対応
